
# diocese_website

Scaffold de site web statique pour un diocèse — livré professionnellement.

## Structure
- `index.html` — Page d'accueil
- `diocese.html`, `paroisses.html`, `pastorale.html`, `actualites.html`, `caritas.html`, `education.html`, `contact.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/logo.svg`, `assets/img/placeholder.jpg`

## Déploiement
Ce site est statique et peut être hébergé sur GitHub Pages, Netlify, Vercel, ou tout hébergement de fichiers statiques.

### Suggestions pro
- Intégrer Leaflet/Google Maps pour la carte des paroisses.
- Ajouter un CMS léger (Netlify CMS / Sanity / Strapi) pour gérer actualités et événements.
- Utiliser Formspree / Netlify Forms / un backend pour le formulaire de contact.
- Mettre en place un workflow CI (GitHub Actions) pour tests et déploiement.
